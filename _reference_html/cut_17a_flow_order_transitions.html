진입점이 두 자리예요 — 대시보드(앱 열기)와 알림(푸시 받기). 자영업 결에선 *내가 능동적으로 열 때*와 *수동적으로 받을 때* 둘 다 자연스러워요.
사진 자리는 자동 이동이 자연스러워요 — 찍자마자 인식 시작. 사용자가 한 번 더 눌러야 하면 결이 끊겨요.
도착점은 매장 잠그기 하나로 깔끔해요. 하루가 명확하게 닫혀요.

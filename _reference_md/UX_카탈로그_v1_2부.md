## 컷 9 · 단계 5 빙산 v3 (가장 큰 변경 자리)

### 메타
- **컷 ID** · catalog_09_stage5_iceberg_v3
- **위치** · 단계 5 진입
- **매크로** · 발견 정리하기
- **결정 차원** · 빙산 세 층 모델 + 한국어 일상어 + 용어 칩

### 시각 구조
- 단계 헤더 — 단계 5 · 진짜 필요 찾기
- 진행 인디케이터 — 단계 5 활성 (단계 4-5 매크로 그룹)
- 좌측 코치 발화 — 빙산 모델 결 안내 (워크북 정신 인용)
- 우측 작업 영역 — 빙산 시각 가로 또는 수직 3 층
  - 1층 (수면 위) — 직접 말한 것 · 사용자가 말로 표현한 자리
  - 2층 (수면 직하) — 행동으로 보여지는 것 · 말로 안 한 행동 패턴
  - 3층 (수면 깊은 곳) — 우리가 분석으로 발견한 것 · 추론으로 도달한 진짜 필요
- 각 층마다 카드 모음 — 단계 3 인용구·단계 4 패턴이 자동 연결
- 가장 깊은 층(Latent)에 *가설 라벨 + 검증 권고*

### 핵심 결정
- 세 층의 *워크북 정본 영어 용어*가 한국어 일상어로 완전 대체 (그룹 A)
- 각 층 라벨 옆 *용어 칩*(작은 i 아이콘) — 클릭 시 인라인 펼침
- 펼침 내용 — 학술 용어(Explicit/Tacit/Latent) + 그룹 라벨(디자인씽킹 워크북 용어) + 짧은 설명 + 우리 한국어 대응 + 이 단계에서 어떻게 발견하나
- 3층 Latent는 *가설* — 사용자가 *맞다고 결정*해야 단계 6 진입 가능
- 단계 4 패턴 카드가 *자동 매핑* — 어느 층에 들어갈지 코치 제안

### 시각 톤 사양
- 빙산 시각 — 수직 3 층 (수면 라인이 1-2층 경계)
- 1층 흰 배경 + 회백 외곽
- 2층 옅은 회백 배경
- 3층 노랑 옅은 배경 #FFFDF4 + 노랑 외곽 (가장 강조)
- 각 카드 — 흰 + 외곽 1px + 인용구 이탤릭 (1층) / 패턴 SemiBold (2층) / 진짜 필요 + 가설 라벨 (3층)
- 용어 칩 — 카드 안 작은 i 아이콘 12px + *용어* 라벨

### v4.1 마이그레이션 메모
- **빙산 세 층 라벨 완전 한국어 대체** (그룹 A) — 마이그레이션 노트 v1.1 §3에서 가장 큰 변경 자리로 명시
  - Explicit → **직접 말한 것** (짧게 *말한 것*)
  - Tacit → **행동으로 보여지는 것** (짧게 *행동에서 본 것*)
  - Latent → **우리가 분석으로 발견한 것** (짧게 *분석으로 발견*)
- 용어 칩 UI 명시 — 카드 내부 i 아이콘 + 클릭 인라인 펼침 (모달 아님)
- 매크로 라벨 — Analyze → **발견 정리하기**
- 진행 인디케이터 — 14 점 패턴
- 검증 컷 — terminology_iceberg_korean_first, terminology_chip_expanded 두 보조 컷이 이 사양을 검증

- 완료 단 — 회백 + 체크 아이콘 + 텍스트 회백
- 미진행 단 — 점선 외곽 + 옅은 텍스트
- 각 단 간 화살표 ↓ 회백
- 코치 *왜?* 질문 작은 회백 + 사용자 응답 차콜

## 컷 12a · 그라운드 룰 펼침 (보완)

### 메타
- **컷 ID** · catalog_12a_ground_rules_expanded
- **위치** · 단계 6 보조
- **매크로** · 아이디어 만들기
- **결정 차원** · 4 룰 자세히 펼침

### 시각 구조
- 컷 12의 그라운드 룰 영역이 *큰 펼침 카드*로 변형
- 4 룰 각각 한 자리씩
  - 양 늘리기 — 일단 많이 만들고 거르기는 나중
  - 비판 보류 — 평가하지 않고 더하기만
  - 비틀어 보기 — 정반대·확대·축소·다른 환경
  - 다른 입장 — 사용자가 아닌 다른 역할로 본다
- 각 룰 카드 안 — 워크북 정신 인용 + 예시 1-2개
- 하단 닫기 버튼

### 핵심 결정
- 그라운드 룰이 *작업 흐름을 막지 않게* 펼침 가능 (필수 아닌 보조)
- 4 룰이 *서로 보완* — 양 + 비판 보류는 한 결, 비틀기 + 다른 입장은 다른 결
- 워크북 정신 *직접 인용* — 이탤릭 노랑 박스

### 시각 톤 사양
- 펼침 카드 — 큰 흰 + 외곽 + 라운드 14px
- 4 룰 — 2x2 그리드
- 각 룰 카드 회백 옅은 배경 + 좌측 노랑 라인 3px
- 워크북 정신 인용 — 노랑 박스 + 이탤릭

### v4.1 마이그레이션 메모
- 매크로 라벨 — Create → **아이디어 만들기**
- 4 룰 한국어 자연스러움 유지

# UX 카탈로그 v1 · 1부 (컷 1-8 · 공감하기 매크로)

> **문서 성격** — 카탈로그 1부의 8 컷을 권위 사양으로 정리. 컷의 결정·구조·시각 사양을 마크다운으로 보존하여, 시각 자체(visualize widget)가 없어도 컷의 DNA를 추적 가능하게.
>
> **작성 시점** — 2026.05.19
> **권위 결정** — 마스터 스펙 v4.1 · 무드 가이드 v3.1
> **마이그레이션** — 이 컷들은 v1 무드 시절 작성되어 v4.1 마이그레이션 필요 (마이그레이션 노트 v1.1 참조)

---

## 사용 안내

각 컷 항목은 5 자리로 구성:
1. **메타** — 컷 ID · 매크로 · 단계 · 결정 차원
2. **시각 구조** — 영역 배치와 레이아웃
3. **핵심 결정** — 이 컷에서 명시된 결정 사항
4. **시각 톤 사양** — 색·폰트·크기 토큰
5. **v4.1 마이그레이션 메모** — 필요한 변경 자리

## 컷 1 · 진입점 선택 화면

### 메타
- **컷 ID** · catalog_01_entry_point_screen
- **위치** · 진입 (단계 0)
- **결정 차원** · 진입점 3종

### 시각 구조
- 페이지 가운데 정렬 (max-width 600px)
- 상단 작은 라벨 — 새 프로젝트
- 큰 헤드라인 — 어디서 출발하세요?
- 부제 한 줄 — 사용자 상태에 따라 진입점 선택
- 진입점 3 카드 수직 배치
  - A · 처음부터 차근차근 — 노랑 외곽 카드 + 가장 많이 라벨
  - B · 아이디어 가진 채로 — 흰 카드
  - C · 프로토타입 빨리 — 흰 카드
- 각 카드 안 — 아이콘 + 제목 + 한 줄 설명 + 시작 단계 안내

### 핵심 결정
- A 진입점 시각 우선 강조 (가장 많이 사용)
- 시작 단계가 카드 하단에 명시 (A · 14 단계 전체 / B · 단계 2부터 / C · 단계 8부터)
- 진입 후 3 질문 진단으로 코칭 강도 결정

### 시각 톤 사양
- 배경 #FAFAF7
- 헤드라인 22px Bold 차콜
- A 카드 배경 #FFFDF4 + 외곽 1.5px solid #F4C430
- B·C 카드 배경 #FFFFFF + 외곽 1px solid #E5E4DD
- 모든 텍스트 Pretendard
- 카드 라운드 12px

### v4.1 마이그레이션 메모
- C 진입점 안내 — 단계 7부터 → **단계 8부터**
- 카드 하단 — *12 단계* → **14 단계**
- 영역 하단 신규 추가 — *진입 후 3 질문 진단으로 코칭 강도 자동 설정*
- 이 컷은 v4.1에서 **로그인 후 홈(§1.5) 영역 4의 일부로 흡수**. 단독 화면이 아닌 홈의 한 영역으로 작동.

- 코치 아바타 노랑 원 + 차콜 아이콘

### 시각 구조
- 상단 진행 인디케이터 (단계 2 활성)
- 단계 헤더 — 단계 2 · 사용자 알아보기 계획
- 좌측 코치 대화 카드 — To-know list 5 카테고리 안내
- 우측 작업 영역
  - 5 카테고리 카드 (목표 · 맥락 · 행동 · 니즈 · 반응)
  - 카드 4(니즈)에 *씨앗 자라남* 시각 — 다른 카드보다 *더 많이 채워진 상태*로 자연스럽게 강조
  - 각 카드 안 코치 제안 후보 2-3개 + 직접 적기 자리

### 핵심 결정
- 5 카테고리가 동등하게 보이되 카드 4 니즈에 미세한 시각 무게 (이게 디자인씽킹의 본질)

- 사용자가 *맞아요/다듬기/다른 자리* 액션으로 답
- 카드 4가 자라나는 결 — 다른 카드는 후보 2개, 카드 4는 3개

### 시각 톤 사양
- 5 카테고리 카드 배경 흰 + 외곽 1px solid #E5E4DD
- 카드 4 미세 강조 — 외곽 1.5px (다른 카드 1px) 또는 노랑 좌측 라인 3px
- 코치 제안 후보 — 노랑 옅은 배경 #FFFDF4 + 노랑 외곽
- 액션 버튼 3종 — 차콜 채움 / 흰 외곽 / 점선 회백

### v4.1 마이그레이션 메모
- 단계 부제 — *To-know list* 그대로 유지 + 용어 칩 추가 (펼치면 *알아둘 자리 목록 · 사용자에 대해 무엇을 모르는지 정리*)
- 매크로 라벨 — Empathize → **공감하기**
- 카드 안 텍스트의 별표(*) 강조 → font-weight 500 또는 색 강조로

### 핵심 결정
- 합성이 *직선적 흐름이 아닌 사다리* — 위아래 이동 가능
- 패턴 발견 후 *어긋나는 신호*를 의식적으로 찾는 자리 (분기 A2 특이값 무시 예방)
- Heavy/Light 분리 가능 (가상 시나리오 안심 매대 결정)
- 패턴이 *가설* 상태에서 단계 5 빙산으로 진입

### 시각 톤 사양
- 합성 사다리 4 층 수직 시각
- 각 층 라벨 작은 회백 letter-spacing
- 패턴 카드 흰 + 외곽
- 어긋나는 신호 카드 — 노랑 외곽 + 노랑 옅은 배경 + 좌상단 *어긋나는 신호* 라벨
- 카드 간 연결선 회백 점선

### v4.1 마이그레이션 메모
- 매크로 라벨 — Analyze → **발견 정리하기**
- 진행 인디케이터 — **14 점** 패턴
- 매크로 그룹 시각 위계 — 인디케이터 점에서 단계 1-3 그룹과 단계 4-5 그룹 사이 시각 분리

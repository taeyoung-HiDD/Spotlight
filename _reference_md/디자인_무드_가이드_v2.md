# 디자인 무드 가이드 v2

> v1의 변경 사항: DesignThinkers Academy의 디자인 톤을 참조해 *두 무드 시스템*으로 분리. 마케팅 영역과 코칭 작업 영역의 결을 명시적으로 다르게 가져간다.
> v1의 핵심 결정(노랑은 통찰에만, 회백색은 양보, 차콜은 결정의 무게)은 작업 영역에서 그대로 유지.

---

## 0. 두 무드 시스템 — 핵심 결정

우리 서비스는 *두 가지 결이 다른 무드*를 함께 사용한다.

**마케팅 무드 (Marketing Mood)** — 랜딩, 소개, 결과, 가격, 사례 페이지. 외부 사용자에게 *서비스를 소개*하는 자리. DesignThinkers Academy 톤을 강하게 적용. **검정 + 노랑 + 큰 세리프 + 강한 사진**.

**작업 무드 (Workspace Mood)** — 12 단계 코칭 흐름, 산출물 보드, 분기 자리, 시스템 영역. *코칭 작업에 몰입*하는 자리. v1 결정 유지하되 *세리프 부분 차용*. **회백색 + 차콜 + 노랑 통찰 강조 + 일부 세리프**.

두 무드 사이의 *전환 자리*가 명시적이다. 사용자가 *마케팅 → 가입 → 진입점 선택*으로 이동할 때 *무드가 부드럽게 전환*된다.

## 1. 마케팅 무드 (Marketing Mood)

### 1.1 컬러 토큰

| 토큰명 | Hex | 역할 |
|---|---|---|
| `--marketing-bg-dark` | `#0A0A0A` | 헤더, 히어로 배경, 강조 섹션 배경 |
| `--marketing-text-on-dark` | `#FFFFFF` | 검정 배경 위 텍스트 |
| `--marketing-accent-yellow` | `#F4C430` | 핵심 강조 한 자리 (HiDD 노랑 유지) |
| `--marketing-bg-light` | `#FFFFFF` | 본문 영역 배경 |
| `--marketing-text-dark` | `#0A0A0A` | 본문 텍스트 |
| `--marketing-bg-subtle` | `#F0EFE9` | 인용구 배경, 통계 카드 배경 |
| `--marketing-text-muted` | `#5A5A56` | 보조 텍스트 |

### 1.2 타이포그래피 — 세리프 + 산세리프 결합

| 토큰 | 폰트 | 크기 | Weight | 용도 |
|---|---|---|---|---|
| `--marketing-hero-serif` | 세리프 | 64px+ | 700 | 히어로 헤드라인 |
| `--marketing-quote-serif` | 세리프 | 32-44px | 400 | 큰 인용구 |
| `--marketing-stat-bold` | 산세리프 | 56-72px | 700 | 큰 숫자 통계 |
| `--marketing-body-sans` | 산세리프 | 16px | 400 | 본문 |
| `--marketing-label-sans` | 산세리프 | 11px | 500 | 라벨, 메뉴 (대문자) |

**세리프 폰트 추천**: Playfair Display, Cormorant Garamond, EB Garamond, 또는 Noto Serif KR (한글). 한글에서는 *명조 계열*. **산세리프**: Inter, Söhne, 또는 Pretendard.

### 1.3 레이아웃 패턴

- **헤더**: 순검정 배경, 흰 메뉴, 핵심 메뉴 하나에만 노랑.
- **히어로**: 대형 사진 또는 영상 + 큰 세리프 헤드라인.
- **로고 그리드**: 검정 배경 흰 로고 / 흰 배경 검정 로고 두 버전 병기.
- **인용구 슬라이더**: 회백 배경 카드 + 큰 세리프 본문 + 작은 인용자 라벨.
- **통계 섹션**: 큰 숫자(`72px+`) + 작은 설명. 네 칸 그리드 패턴.
- **사례 그리드**: 전면 큰 사진 + 한 브랜드 로고 overlay.

### 1.4 적용 화면 (예정)

- 랜딩 페이지 (메인)
- 코스 소개 페이지 (12단계가 무엇인지)
- 결과·사례 페이지 (사용자 후기, 프로젝트 결과)
- 가격·플랜 페이지
- 소개·미션 페이지

## 2. 작업 무드 (Workspace Mood)

v1의 결정을 유지하되 *세리프 차용*과 *전환 자리*를 추가한다.

### 2.1 컬러 토큰 — v1 그대로

| 토큰명 | Hex | 역할 |
|---|---|---|
| `--workspace-charcoal` | `#2D2D2A` | 메인 텍스트, 액션 버튼, 헤더 |
| `--workspace-insight-yellow` | `#F4C430` | 통찰 강조 (자제하여 사용) |
| `--workspace-yellow-tint` | `#FCF3D6` | 통찰 칩 배경 |
| `--workspace-yellow-deep` | `#8B6914` | 노랑 위 텍스트 |
| `--workspace-yellow-bg` | `#FFFDF4` | 통찰 카드 내부 배경 |
| `--workspace-off-white` | `#FAFAF7` | 페이지 배경 |
| `--workspace-white` | `#FFFFFF` | 카드 본체 |
| `--workspace-warm-gray` | `#E5E4DD` | 테두리 |
| `--workspace-muted-gray` | `#6B6A66` | 보조 텍스트 |
| `--workspace-light-gray` | `#A09E94` | 힌트 |
| `--workspace-bubble-coach` | `#F8F7F1` | 코치 발화 배경 |
| `--workspace-bubble-user` | `#F3F2EC` | 사용자 발화 배경 |

### 2.2 타이포그래피 — 산세리프 기본 + 세리프 부분 차용

v1의 산세리프 사이즈는 그대로 유지. 추가로 *세리프가 들어올 세 자리*가 명시된다.

**세리프 적용 자리**:

| 자리 | 폰트 | 크기 | 용도 |
|---|---|---|---|
| `--workspace-stage-title-serif` | 세리프 | 18-22px | 단계 헤더 (예: "단계 5 · Needs Definition") |
| `--workspace-coach-quote-serif` | 세리프 | 14-16px | 코치가 워크북 정신 인용할 때 |
| `--workspace-completion-serif` | 세리프 | 24-32px | 슬롯·단계 완료 축하 화면 |

다른 모든 자리(본문, 카드, 라벨, 메타)는 *산세리프 그대로*. 세리프는 *특별한 순간*에만 등장.

### 2.3 v1 그대로 유지되는 결정들

- **노랑 사용 규칙** — 통찰의 한 자리에만 (15초 핵심 원칙 5번 유지)
- **분할 화면 패턴** — 대화 + 산출물 보드
- **카드 상태 4종 + 확장 4종** — v1 §4 그대로
- **코치 모드 6종 + 아바타 시각화** — v1 §4 그대로
- **8px grid spacing** — v1 §3 그대로
- **Border Radius 6-16px 시스템** — v1 §3 그대로
- **시맨틱 → 시각 매핑** — v1 §4 그대로
- **출처 추적 칩의 파란 톤 예외** — v1 §4 그대로

### 2.4 새로 추가되는 자리

**완료 축하 화면 — 검정 톤 인터스티셜**. 슬롯·단계·전체 프로젝트 완료 시 *짧은 검정 화면*으로 전환. 검정 배경 + 노랑 강조 + 큰 세리프 메시지. 마케팅 무드의 결을 *작업 영역에 짧게 차용*하는 자리. 사용자의 *성취 순간*을 강조하는 호흡.

**전환 인터스티셜 — 마케팅 → 작업**. 진입점 선택 후 첫 번째 작업 화면 전 *2-3초 전환 자리*. 검정 톤이 회백색으로 *부드럽게 페이드*. 사용자가 *마케팅 사이트에서 작업 도구로 들어오는 자리*를 시각적으로 표현.

## 3. 두 무드 사이의 전환

### 3.1 전환 자리 정의

- **마케팅 → 작업**: 진입점 선택 후, 검정 인터스티셜 → 회백색 페이드.
- **작업 → 마케팅**: 사용자가 로그아웃 또는 *내보내기* 누를 때, 회백색 → 검정 인터스티셜.
- **작업 내부 강조 자리**: 완료 축하 화면이 *짧은 검정 호흡*으로 등장.

### 3.2 전환 디테일

전환은 *2-3초 이내*에 끝난다. 사용자가 *모드가 바뀐다*는 신호를 받되 *기다리는 느낌은 없어야* 한다. 검정 인터스티셜은 *큰 세리프 메시지 한 줄 + 노랑 점 하나*로 단순하게.

---

## 4. 15초 핵심 원칙 (v2 업데이트)

새 화면을 만들기 전에 다음 다섯 원칙을 머리에 둔다.

1. **어느 영역인가 — 마케팅 vs 작업** — 두 무드 시스템 중 어디에 속하는지 먼저 결정.
2. **노랑은 통찰의 순간에만** — 두 무드 모두 공통. 노랑이 너무 많으면 의미가 사라진다.
3. **세리프는 인간 중심·창의적 결 — 산세리프는 구조** — 마케팅에서는 둘이 강하게 결합, 작업에서는 *특별한 순간에만* 세리프.
4. **마케팅 = 강한 대비 + 큰 사진 / 작업 = 절제된 회백 + 작은 디테일** — 두 무드의 정체성.
5. **한 화면에 통찰의 자리는 한 곳** — v1 그대로 유지.

## 5. v1과 v2의 차이 요약

| 자리 | v1 | v2 |
|---|---|---|
| 컬러 시스템 | 회백·차콜·노랑 | 작업 무드는 그대로 + 마케팅 무드 추가 (검정·노랑·세리프) |
| 타이포 | Medium 산세리프만 | 작업은 산세리프 기본 + 세 자리 세리프 차용 / 마케팅은 세리프 + 산세리프 결합 |
| 화면 영역 | 작업 화면 중심 | 마케팅 영역 + 작업 영역 명시적 분리 |
| 전환 자리 | 정의 안 됨 | 마케팅 ↔ 작업 전환 인터스티셜 명시 |
| 검정 사용 | 차콜 #2D2D2A만 | 마케팅은 순검정 #0A0A0A, 작업은 차콜 그대로 |
